# 🟢 Kinetic Actions

**Kinetic Actions** is a high-performance, premium project management interface designed for clarity, speed, and deep analytical insight. Built with a "Terminal-First" philosophy, it combines the minimalist aesthetics of a Mac desktop with the robust task-tracking power required for modern software development.

## 🚀 Key Features

### 🖥️ High-Fidelity Workspace

- **Intuitive Project Switcher**: Seamlessly toggle between multiple projects with a keyboard-friendly selection modal.
- **Mac-Style Interface**: Enjoy a centered global search, glassmorphic UI elements, and a choice between **Stunning Dark** and **Sharp Light** modes.
- **Persistent State**: Your workspace stays exactly how you left it, thanks to robust `localStorage` integration.

### 📊 Advanced Project Tracking

- **Kanban Workflow**: Drag-and-drop task management with automated activity logging.
- **Project Pulse & Health**: Real-time SVG charts visualizing task velocity, priority distribution, and overall project stability.
- **Integrated Roadmap**: Chronological view of project activity and upcoming deadlines.

### ⌨️ Developer-Centric Tools

- **Global Command Search**: Hit `F` or `/` to search across all tasks, projects, and metadata instantly.
- **Audit Trail**: Every change is tracked in a secure Activity Log for full accountability.
- **Keyboard First**: Native shortcuts for creating tasks (`N`), switching views, and toggling themes.

## 🛠️ Technology Stack

- **Core Logic**: Vanilla JavaScript (ES6+) for maximum performance.
- **Styling**: Custom CSS3 Variable-driven architecture.
- **Icons**: Hand-crafted SVG icon set.
- **Persistence**: Browser LocalStorage.

## 🏁 Getting Started

1. Clone the repository.
2. Open `index.html` in any modern browser.
3. No build steps or dependencies required—just pure, kinetic power.
