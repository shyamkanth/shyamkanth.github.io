How it works

Up and recording in 3 clicks

No setup. No tutorial. Just install and start capturing.

1

Install the extension

Add OpenRec to Chrome from the Web Store in one click. No sign-up, no permissions beyond what's needed.

2

Choose your source

Pick screen, camera, or both. Select your audio source — mic, system audio, or a mix. Customize the webcam bubble if you like.

3

Record & save locally

Hit record. When you're done, the video saves straight to your browser. Trim it, export as GIF, or download the .webm file.
